# Basic Linux Commands

## pwd
### Description:
Displays the current directory you are in.

### When to use:
When you are lost in the filesystem.

### Command:
pwd

---

## ls
### Description:
Lists files and directories.

### Command:
ls
ls -la

### When to use:
To see files, including hidden ones.

---

## cd
### Description:
Changes directory.

### Command:
cd /home/kali/Desktop

### When to use:
To move between folders.

---

## clear
### Description:
Clears terminal screen.

### Command:
clear
